# 青龙订阅配置说明

## 📅 定时任务配置

### 方式一：脚本头部添加 cron（推荐）

在脚本开头添加 `@cron` 注释，青龙会自动识别：

```javascript
/**
 * @cron 51 9 * * *
 */
```

或：

```python
# @cron 0 9 * * *
```

### 方式二：使用 cron.yml 配置文件

在仓库根目录创建 `cron.yml` 文件：

```yaml
# 格式：脚本路径：cron 表达式
js/juejin_v3.js: 51 9 * * *
python/baidu_tieba.py: 0 9 * * *
```

### 方式三：手动配置

在青龙面板 → 定时任务 中手动修改每个任务的定时规则。

## 📋 推荐定时规则

### 掘金签到
```
51 9 * * *    # 每天 9:51（避开高峰期）
```

### 百度贴吧
```
0 9 * * *     # 每天 9:00
```

### 百度网盘
```
0 8 * * *     # 每天 8:00
```

### Google Voice 保号
```
0 0 */15 * * *    # 每 15 天 0:00
```

## 🔧 青龙订阅拉取命令

```bash
# 基础拉取（扫描所有 js/py 文件）
ql repo https://github.com/LceAn/qinglong-scripts.git

# 只拉取指定文件（推荐）
ql repo https://github.com/LceAn/qinglong-scripts.git "js/|python/" "" "lib/"

# 参数说明：
# 第 1 个参数：仓库地址
# 第 2 个参数：白名单（只扫描这些文件/目录）
# 第 3 个参数：黑名单（不扫描这些文件/目录）
# 第 4 个参数：依赖文件（不创建任务）
```

## ⚠️ 注意事项

1. **lib/ 目录** 是依赖模块，不会被创建为定时任务
2. **cron.yml** 是配置文件，不会被创建为定时任务
3. 脚本中的 `@cron` 注释只在首次添加任务时生效
4. 如需修改定时规则，建议在青龙面板手动修改
