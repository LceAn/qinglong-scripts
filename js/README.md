# 掘金签到脚本 v3.0

## 📋 功能说明

- ✅ **自动签到** - 每日自动签到获取积分
- ✅ **免费抽奖** - 使用免费次数抽奖（矿石/道具）
- ✅ **挖矿游戏** - 自动完成挖矿小游戏
- ✅ **沾喜气** - 沾好友喜气获取额外奖励
- ✅ **多通道通知** - 邮件/钉钉/企业微信 三种通知方式
- ✅ **智能重试** - 网络请求失败自动重试
- ✅ **执行统计** - 记录执行耗时和积分变化

## 🔧 配置方法

### 1. 青龙环境变量

在青龙面板添加以下环境变量：

| 变量名 | 必填 | 说明 | 默认值 |
|--------|------|------|--------|
| `JUEJIN_COOKIE` | ✅ | 掘金 Cookie（从浏览器获取） | - |
| `JUEJIN_ENABLE_MAIL` | ❌ | 启用邮件通知 | `true` |
| `JUEJIN_ENABLE_DINGTALK` | ❌ | 启用钉钉通知 | `true` |
| `JUEJIN_ENABLE_WXWORK` | ❌ | 启用企业微信通知 | `true` |
| `JUEJIN_RETRY_TIMES` | ❌ | 请求重试次数 | `3` |
| `JUEJIN_RETRY_DELAY` | ❌ | 重试延迟 (毫秒) | `1000` |

### 2. 获取 Cookie

1. 打开浏览器访问 https://juejin.cn/
2. 登录账号
3. 按 F12 打开开发者工具
4. 进入 Network 标签，刷新页面
5. 找到任意 API 请求（如 `api.juejin.cn`）
6. 复制 Request Headers 中的 `cookie` 字段完整内容
7. 填入青龙的 `JUEJIN_COOKIE` 环境变量

### 3. 定时任务

```bash
# 每天 9:51 执行
51 9 * * * juejin_v3.js
```

## ⚠️ 注意事项

1. **Cookie 有效期** - 掘金 Cookie 会过期，建议定期更新（1-2 个月）
2. **频率限制** - 不要设置过于频繁的执行时间
3. **账号安全** - 妥善保管 Cookie，不要泄露
4. **通知配置** - 如不需要某种通知，设置对应环境变量为 `false`

## 🐛 常见问题

### Q: 提示 "cookie 不能为空"
A: 检查青龙环境变量 `JUEJIN_COOKIE` 是否正确设置

### Q: 签到失败，提示 API 错误
A: Cookie 可能已过期，请重新获取

### Q: 通知发送失败
A: 检查对应的通知模块（sendMail/sendDingTalk/sendWxWork）是否正确配置

### Q: 如何禁用某个通知通道？
A: 设置对应环境变量为 `false`，例如：
```bash
JUEJIN_ENABLE_MAIL=false
```
